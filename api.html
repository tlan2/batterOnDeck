MLB Data API

Batter On Deck uses promises to gather and fetch data from MLB Data API to supply our website with player info. We used 3 of the main API calls to pull players' personal, seasonal and career data. Examples are provided below. We had to mix and match them in order to retrieve all the needed information. You can learn more about the other API calls at appac.github.io/mlb-data-api-docs or rapidapi.com/theapiguy/api/mlb-data/endpoints.
